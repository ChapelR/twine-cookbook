:: StoryTitle
Fairmath in Snowman

:: Start
<%

function increase(x,y) {
	return Math.round(x+((100-x)*(y/100)));
}
	
function decrease(x,y) {
	return Math.round(x-(x*(y/100)));
}

%>

Decrease 100 by 50% using Fairmath:
<%= decrease(100, 50) %>

Increase 50 by 50% using Fairmath:
<%= increase(50, 50) %>
