# Snowman HTML

## Snowman 1.X HTML

```html
<body>
  <div id="main">
    <div class="passage" aria-live="polite">
      <p>Double-click this passage to edit it.</p>
    </div>
  </div>
</body>
```

```html
    </tw-passage>
  </tw-story>
</body>
```
