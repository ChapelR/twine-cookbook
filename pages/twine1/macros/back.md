# `<<back>>`

!!! Warning
    You are reading documentation for Twine 1. This is maintained for archival purposes only.

The `<<back>>` macro provides a "‚Üê back" link and will, upon being clicked, either move backward one passage in the history of the story or, when given a parameter, will go back to that exact passage in the history of the story.

(See also: `<<return>>` macro )

## Previous Passage Example

```twee
<<back>>
Going back to "Start" Passage Example:
<<back "Start">>
```
